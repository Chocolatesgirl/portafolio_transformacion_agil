# 🧩 Casos Prácticos — Aplicación de Kanban en Productos Digitales

## 🎯 Propósito

Esta carpeta presenta la aplicación práctica de Kanban como sistema de gestión de flujo en el desarrollo de productos digitales.

El foco no está en la teoría, sino en:

* cómo se comporta el sistema en operación real
* qué problemas emergen en el flujo
* qué decisiones se toman para estabilizarlo
* qué impacto tienen esas decisiones

👉 Cada caso refleja gestión de delivery basada en datos y flujo.

---

## 🚀 Caso destacado

### 📱 Implementación de Kanban en App de Inglés para Niños

📄 `caso_practico_kanban_app_ingles.md`

---

## 🧩 Contexto del caso

Desarrollo de una:

👉 **Aplicación de aprendizaje de inglés para niños de 3 a 8 años**

Objetivos del producto:

* mejorar retención de usuarios
* aumentar engagement mediante experiencias interactivas
* optimizar aprendizaje mediante audio, juegos y recompensas

---

## 🚨 Problema abordado

El equipo presentaba:

* sobrecarga de trabajo en desarrollo
* acumulación en Testing y QA
* falta de visibilidad del flujo
* baja predictibilidad en entregas

👉 El sistema operaba de forma reactiva, generando cuellos de botella.

---

## 🧠 Enfoque aplicado

Se implementa Kanban como sistema de gestión de flujo, integrando:

* diseño de flujo end-to-end
* políticas WIP para control de capacidad
* operación basada en flujo pull
* métricas para toma de decisiones

👉 El foco se desplaza de gestionar tareas a gestionar el sistema de trabajo.

---

## 🛠️ Principales decisiones

* implementación de límites WIP
* priorización de tareas en progreso
* control de entrada de trabajo
* redistribución de capacidad del equipo
* fortalecimiento de prácticas de testing

---

## 📊 Resultados obtenidos

* reducción del Cycle Time (~30%)
* mayor estabilidad del flujo
* disminución de cuellos de botella
* incremento del throughput
* mejora en la predictibilidad

---

## 📈 Impacto en el producto

* mayor tiempo de uso por sesión
* mejor experiencia de aprendizaje
* mayor interacción con funcionalidades
* feedback positivo de usuarios
